## xpose出入
> 目前实现了一个自定义微信地区的功能

## 使用工具
* 方式一
    1. root手机
    2. 安装xpose框架
    3. 安装本APP
    4. 重启后生效

* 方式二
    1.安装[VirtualXposed](https://github.com/android-hacker/VirtualXposed/releases)
    2.[VirtualXposed使用方法](https://www.jianshu.com/p/8cb84bad1e7f)按照该方法及微信及本App添加到VirtualXposed中,启用本App模块并重启
